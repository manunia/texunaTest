    Генератор текстовых отчётов.

Генерируется текстовый отчет. Отчет разбивается на страницы заданной высоты. Страницы отделяются строкой, содержащей один символ '~'.
Для перехода на новую строку использован системный разделитель строк.
Заголовок и данные в строке отделяются символом '|' и выравнены по левому краю. Слева и справа от значения всегда есть по одному пробелу.
Ширина колонок имеет заданное значение и не включает в себя пробелы слева и спарва от значения.
Пробелы заполняют неиспользованное место.
Заголовок повторяется на каждой странице. Строки отделяются строкой символов '-'.
Если значение не умещается в отведенное место, то оно переносится на следующую строку.

На вход принимается файл настроек settings.xml и файл с данными source-data.tsv.
Используется язык Java.
Для получения данных из xml использована бибилиотека univocity-parsers.

Результат записывается в файл repotr.txt в кодировке UTF-16.
